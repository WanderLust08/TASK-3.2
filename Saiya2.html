Kaioken Goku

About :

  • Kaioken Is Goku's level one super attack. By manipulating their body, the user can theoretically increase their energy an infinite amount. In practice, this is impossible, as the Kaioken puts immense stress on the body.
  • Kaioken is one of the most complicated attacks in the entire game, as it essentially gives the user a set number of inputs to perform a custom Kaioken-combo.
  • These moves can be used one after the other, combo-ing into each other. Keep in mind that all combinations of these moves will not connect.
  • Kaioken finisher does not count as one of these inputs, but can only be used once, as using it will end the attack
  • Therefore, after performing all available inputs, it is still possible to use Kaioken Finisher, or Kaioken Kamehameha.
